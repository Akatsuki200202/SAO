# SAO: Supplemental Material for ICDE 2027

This repository provides the supplemental material for **SAO**, submitted to **ICDE 2027**. SAO is a single-attacker backdoor attack method for Graph Federated Learning.

## Dataset Preparation

Users can use the tools provided under the `data/` directory to download and split the required datasets. The processed datasets will be saved under the `datasets/` directory.

## METIS Partition

For the code related to METIS-based graph partitioning, please refer to:
[https://github.com/james77777778/metis_python](https://github.com/james77777778/metis_python)
## Quick Start

To quickly run SAO, please try:

```bash
python main.py
```
